# ChitChat

A modern, dark-themed login and dashboard interface for a real-time chat application.

## Features
* **Night Mode:** Sleek dark blue color scheme with terminal green accents.
* **Responsive Design:** Styled for mobile and desktop screens.
* **Navigation:** Built-in transition between login and dashboard pages.
